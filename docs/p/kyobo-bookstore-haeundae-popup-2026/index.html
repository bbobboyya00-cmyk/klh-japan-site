2026年5月、海雲台が文学の香りに包まれる理由

入口のモダンな建物に掲げられた大きなバナーが、遠くからでも「あ、あそこだ！」ってすぐに分かります。中に入る前からなんだかワクワクしちゃうデザインで、やっぱり現場に来るとテンションが上がりますね（笑）

入口には可愛いピンクのベンチが置いてあって、ここですでに写真を撮っている人が結構いました。単なる本屋さんの入口っていうより、おしゃれな展示会の入り口みたいな雰囲気で、掴みはバッチリです。

教保文庫ポップアップスクールの楽しみ方とは

中に入ると、教保文庫らしい広々とした空間に本がぎっしり並んでいて、やっぱり落ち着きますね。今回のポップアップはデジタルとアナログの融合がテーマらしく、本好きにはたまらない仕掛けが随所に隠されています 📚

特にこの黄色い壁のエリアにあるデジタルキオスク、これが意外と優秀なんです。今の気分に合わせて本を勧めてくれるんですけど、これがまた「分かってるな〜」という選書で（笑）

あと、教保文庫といえば雑貨も外せませんよね。棚に綺麗に並んだ文房具を見ているだけで、ついつい財布の紐が緩みそうになります（というか、実際いくつか買っちゃいました）。

詩の心に触れる、金洙暎（キム・スヨン）ワークショップ

今回の目玉の一つが、韓国の国民的詩人、金洙暎（キム・スヨン）さんの展示コーナーです。モノクロの肖像画が飾られた空間は、そこだけ少し空気がピリッと引き締まっていて、文学の深みを感じさせてくれます。

実際のワークショップ会場も覗いてみましたが、まるで本当の教室みたいで懐かしい気分になります。ここで詩を写したり朗読したりするんですけど、大人になってからの「学び」って、意外と楽しいんですよね 🖊️

さらに奥に進むと、まるで映画のセットみたいな重厚な木の書庫が現れます。ここは本当に静かで、海雲台の賑やかさを忘れて読書に没頭するには最高の穴場スポットかもしれません。

ハンギョドンと釜山の出会い、限定グッズの魅力

オレンジ色の暖かい照明に照らされた本棚の列、ここが今回の「映え」ポイントです。左右対称に並んだ本がどこまでも続いているようで、広角レンズで撮るとめちゃくちゃ綺麗に写りますよ（笑）

そして最後に、サンリオのハンギョドン！釜山の海をイメージしたパステルブルーの壁に、お馴染みのゆるいイラストが描かれていて、思わず「可愛い…」って声が出ちゃいました 💙

2026年限定のポスターも、ハンギョドンが釜山を満喫しているデザインで特別感がすごいです。文学の香りとキャラクターの可愛さが絶妙に混ざり合った、海雲台でしか味わえない贅沢な空間でした。
