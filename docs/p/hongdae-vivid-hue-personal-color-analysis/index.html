弘大のVivid Hueで体験するパーソナルカラー診断

スタジオに入ると、色とりどりのカラーチップが綺麗に整頓されており、まるでアート空間のような雰囲気が漂っています。🎨 まずはカウンセリングシートに、普段よく着用する服の色や、どうしても肌から浮いてしまうと感じる色、好みのメイクスタイルを細かく書き込んでいきます。自分の好みを客観的に整理するだけでも、新しい発見があります。

丁寧なドレープ診断と詳細なトーン分類の流れ

診断スペースには、顔色を正確に捉えるためのプロ仕様 LED照明と、数多くの色布（ドレープ）が並べられています。鏡の前に座り、様々な色のドレープを1枚ずつ顔の下にあてていく作業が始まります。今回のパーソナルカラー診断では、イエローベース（春・秋）とブルーベース（夏・冬）の分類はもちろん、さらに細かくトーン（ライト、ソフト、ミュート、ブライト、ダーク、ペール）を分析していきます。

実際に布をあてて比較することで、自分を最も引き立てるベストカラーと、避けるべきワーストカラーが明確になります。ドレープを1枚変えるだけで、顔の印象や肌の明るさが変化する様子をしっかりと確認できます。

このように、細かい色の違いによって肌のトーンがどのように変化するかを、プロの目で一対一で丁寧に見てくれます。素材ごとの相性も教えてもらえるため、デニムやツイードなど、毎日のコーディネートにすぐ取り入れられる実用的なアドバイスが特徴です。

ポーチ診断と日常に活かせるスタイリング提案

診断の後は、手持ちのメイクポーチの中身をチェックしてもらう時間です。💄 普段使用しているリップやアイシャドウが自分のトーンに合っているかを1つずつ確認し、足りないカラーやおすすめのメイクアイテムをその場で提案してくれます。

パーソナルカラーの基本理論についても、タブレットを使って視覚的に説明してくれるため、初心者でもスムーズに理解できます。感覚的な部分だけでなく、論理的な解説があるため、非常に納得感があります。

スタジオ内にある Vivid Hue のブースには、国内外の様々なコスメが揃っており、自分に合うリップやチークをその場で実際に試すことができます。さらに、似合う髪色やアクセサリーのゴールド・シルバーの比較まで、トータルでアドバイスが受けられます。

手の甲にアクセサリー用の素材をあてて、肌が綺麗に見える金属の質感を見極めていきます。普段ゴールドばかり選んでいた場合でも、シルバーの方が肌が澄んで見えるといった新たな気づきが得られることもあります。
