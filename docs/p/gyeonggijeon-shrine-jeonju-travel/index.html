慶基殿（キョンギジョン）とは？朝鮮王朝の歴史が息づく神聖な空間

全州韓屋村の中心に位置する慶基殿は、1410年に朝鮮王朝の創始者である太祖・李成桂の肖像画（御真）を祀るために建てられた由緒ある場所です。長い歴史の中で一度焼失したものの、1614年に再建され、今もその姿を留めています。 本殿である正殿に向かう際は、正面から全体を捉えるほか、少し斜めの位置から撮影すると、木造建築の奥行きが綺麗に表現できます。

参道を歩く前に知っておきたい参拝の作法とルート

境内に入ると、まず目に入るのが神聖なエリアの入り口を示す赤い「紅箭門（ホンサルムン）」です。参拝の基本は東入西出、つまり右側の門から入り、左側の門から出るのがマナーとなっています。 また、中央の少し高くなっている石畳の道は「神道（シンド）」と呼ばれ、神様（太祖の魂）が通る道とされているため、中央の道は歩かないように避けて歩くのがマナーです。

本殿を囲む回廊では、伝統的な木の柱が額縁のようになり、切り取られた中庭の景色が絵画のように見えます。柱と柱の間に立って撮影すると、奥行きのあるポートレートが撮影できます。

四季折々の美しさと歴史を保管する全州史庫の佇まい

春や夏には青々とした緑が美しい慶基殿ですが、冬に訪れると、瓦屋根に白い雪が積もった静寂な世界が広がります。❄️季節ごとに異なる表情を見せてくれるのも魅力です。

さらに奥へ進むと、朝鮮王朝の実録を保管していた「全州史庫（チョンジュサゴ）」が見えてきます。周囲の木々と調和した木造の楼閣は、歴史の重みを感じさせる静かな佇まいをしています。

精巧な木造建築の… -> 精巧な木造建築のディテールとおすすめの撮影アングル

建物の細部に目を向けると、梁や柱の色彩、精密に組み上げられた軒下の造形が見られます。職人たちの高度な技術が、何百年もの時を超えて今に伝えられているのを感じられます。

少し高い位置から見上げる楼閣と、なだらかな曲線を描く瓦屋根の重なりは、伝統建築の美しさそのものです。🏯この背景を活かすなら、周辺でハンボク（伝統衣装）をレンタルして散策するのも方法の一つです。

歴史の重みを感じながら、ゆっくりと境内を散策してみてはいかがでしょうか。全州を訪れる際は、一度候補に入れてみてもよさそうです。

  • 観覧時間: 9:00〜19:00（季節により変動あり、年中無休）
  • 入場料: 大人 3,000ウォン（カード決済可能）
