ソンス洞が紫に染まる！ポケモン30周年記念「メタモン遊び場」徹底解説

K-Life HackディレクターKが、ソウル・ソンス洞に登場したポケモン30周年記念ポップアップ「メタモン遊び場」を徹底解説。限定グッズや工業的な街並みに溶け込む空間演出を深掘りします。

K-Life HackディレクターのKです。2026年4月23日、ソウルのトレンド発信地である聖水洞（ソンスドン）は、かつてない熱気に包まれています。

今回は、日本が生んだ世界的IP『ポケットモンスター』の生誕30周年を祝し、ソウルで最も「ヒップ」な場所として知られる聖水洞に誕生した特別な空間、**「メタモン遊び場（Metamong Playground）」**をディレクターの視点で紐解いていきます。

韓国では「メタモン（Metamong）」の名で親しまれるへんしんポケモン、メタモン。その自由自在な変身能力が、聖水洞の古い工場跡地と出会い、いかなる化学反応を起こしたのか。単なるキャラクターイベントを超えた、没入感溢れるエディトリアルな体験をお届けします。

聖水洞「メタモン遊び場」：Kディレクターからの最速ガイド -聖水洞のインダストリアルな空間を舞台に、ポケモン30周年を記念した体験型ポップアップ「メタモン遊び場」が期間限定で開催。メタモンの「へんしん」能力をテーマにした独創的な空間演出と、世界中のコレクターが注目する「ポケピア（Poképia）」限定グッズの販売が最大の目玉です。ソウルの若者文化とグローバルなIPが融合した、2026年上半期で最も注目のポップアップといえるでしょう。

工業的な美学と「メタモン」の意外な親和性💜

聖水洞（ソンスドン）は、かつての靴工場や倉庫がリノベーションされ、洗練されたカフェやショップに生まれ変わった「韓国のブルックリン」です。今回の「メタモン遊び場」は、その地の利を最大限に活かし、無骨な赤レンガやコンクリートの質感と、メタモンの柔らかく鮮やかな紫色のコントラストを際立たせています。

メタモンの持つ「決まった形を持たない（Amorphous）」という特性は、絶えず変化し続ける聖水洞の街のアイデンティティと見事に呼応しています。訪れる人々は、かつての工場の面影を残す空間の中に、突如として現れたメタモンたちの遊び場に足を踏み入れることで、日常が非日常へと「へんしん」する瞬間を体験することになります。

没入型フォトゾーン：SNS時代の体験価値

会場内には、30周年記念ロゴを配した巨大なターンテーブル型のオブジェや、メタモンが変身した様々なポケモンのインスタレーションが配置されています。これらは単なる展示物ではなく、訪問者が自由に撮影し、自らのストーリーをSNSで共有することを前提に設計されています。

特に、工業的な背景とメタモンのキッチュな可愛らしさが同居するフォトゾーンは、デジタルネイティブ世代にとって「インステジェニック（Instagenic）」な魅力に溢れています。一歩踏み込むたびに新しい発見があるこの空間は、30年にわたり愛され続けてきたポケモンの歴史に、聖水洞という現代的な解釈を加えた、まさに「文化の変身」といえるでしょう。

コレクター必見！限定グッズ「ポケピア」の誘惑🛍️

ファンの間で最大の話題となっているのが、本イベント限定のグッズラインナップ、「ポケピア（Poképia）」シリーズです。ピカチュウやカビゴンなど、おなじみのポケモンたちがメタモンの独特な表情で表現されたこのシリーズは、単なるキャラクター商品を超えた「現代のアートピース」としての風格さえ漂わせています。

韓国国内でも非常に高い人気を誇るため、1人あたりの購入制限が設けられているにもかかわらず、連日早期の完売が予想されます。メタモン特有の、どこか間の抜けた、しかし愛くるしい表情を纏ったポケモンたちは、コレクターの所有欲を強く刺激します。ぬいぐるみだけでなく、日常生活を彩るリビング雑貨やステーショナリーまで、幅広いラインナップが展開されています。

訪問を計画される方へ：ディレクターのアドバイス

「メタモン遊び場」は、ソウル地下鉄2号線の**「聖水（ソンス）駅」4番出口**から徒歩約10分という、非常にアクセスの良い場所に位置しています。しかし、聖水洞の街全体がそうであるように、駐車場を確保するのは至難の業です。近隣の有料駐車場を利用するよりも、ソウルの優れた公共交通機関を利用することを強くお勧めします。

また、混雑を避けてゆっくりと空間を楽しみたい方は、平日の午前中を狙って「オープンラン」を計画するのが最善です。限定グッズの在庫状況や現場の待機列に関する最新情報は、ポケモンコリア（Pokémon Korea）の公式Instagramを随時チェックし、スマートな訪問を心がけてください。
