ザ・ヒュンダイ・ソウルで見つける、韓国伝統菓子の新境地「ハンジョンソン」

「ザ・ヒュンダイ・ソウル」地下1階にある「ハンジョンソン」は、ミニマルな木製デザインと韓国の美しい月壺が並ぶファサードが本当に目を引くんです。まるでギャラリーみたいで、お店に入る前からワクワクが止まりません。私も昨年初めて訪れた時、その洗練された空間にすっかり魅了されちゃいましたよ！✨

店内に一歩足を踏み入れると、色とりどりの伝統的な紙に包まれた、まるで芸術品みたいなフルーツ餅がずらりと並んでいて、もうテンション爆上がりです！一つ一つ手作業で作られた繊細な美しさは、見ているだけで幸せな気分になっちゃいますよ。フレッシュな果物の瑞々しさと、お餅の組み合わせは想像以上のおいしさで、一度食べたら忘れられません!! 😋

モダンなデザインと伝統的なおもてなしが融合した「ハンジョンソン」のブティックは、「ザ・ヒュンダイ・ソウル」の洗練された雰囲気にもぴったりなんです。私は平日の午前中に行ったので、ほとんど待たずに買えましたが、やっぱり人気店だから週末はちょっと並ぶかも。(でも、スタッフさんの手際が良いから大丈夫ですよ！)

大切な人への贈り物には、この美しいシルク製のポッチュモニ（福袋）が絶対おすすめ！🎁 色も素材も豊富で、プレミアムなギフトラッピングとして選べるんです。私も昨年、友達へのプレゼントにいくつか買ったら、その上品さにすごく喜んでくれましたよ。手土産にも最高なので、ぜひチェックしてみてほしいです。

ブランドのこだわりって、本当に細部まで行き届いてるんです。繊細なピンクのフローラルイラストと「ハンジョンソン」のロゴが入ったショッピングバッグは、もう可愛すぎて持ち歩くだけで気分が上がっちゃいますよね！昨年もあまりの可愛さに、このバッグ目当てで買いすぎちゃいました。(可愛すぎて、私もたくさん買っちゃいましたよ！)。2026年の新しいデザインも、今からすごく楽しみです！😍

メニューボードには、新鮮なイチゴから黄金のパイナップルまで、フルーツ餅のラインナップが芸術的なイラストで紹介されてるんです。どれも美味しそうで迷っちゃいますよね！昨年はメロン、マンゴー、イチゴの3種類を試したんですが、どれも果物本来の甘さとヨーグルトの酸味が絶妙にマッチしてました。特にメロンとマンゴーは、口いっぱいに広がる果汁が最高でしたよ！🍈🥭

ザ・ヒュンダイ・ソウル：ショッピングを超えた感動体験

「ザ・ヒュンダイ・ソウル」は、その建物自体がヨイドの景色を変えるアート作品なんです。象徴的な赤いクレーンと白い格子構造が織りなす壮大なファサードは、遠くからでも一目でその存在感がわかりますよね。訪れるたびに、この場所の洗練された雰囲気に圧倒されちゃいます。

デパートの中に広がる熱帯のオアシス「サウンズフォレスト」は、自然光が降り注ぐガラスの天井の下で、手入れの行き届いた植物たちが心を癒してくれるんです。ショッピングの合間にここで一息つけば、まるで都会の喧騒から離れた別世界にいるみたい。(最高の癒しスポットです！)。私も昨年、ここで素敵な写真をたくさん撮りましたよ。📸

12メートルもの高さから水が流れ落ちる「ウォーターフォールガーデン」は、ダイナミックなキネティック彫刻がデパート内に穏やかな雰囲気を作り出してくれます。水音を聞きながら都会の真ん中で自然を感じられるなんて、まさに「ザ・ヒュンダイ・ソウル」ならではの魅力ですよね！🌊

特にホリデーシーズンには、アトリウムに広がるヨーロッパ風のクリスマスヴィレッジが、お祭りみたいに賑わうんです！🎄 昨年も心温まる木製の小屋や常緑樹が飾られてて、まさにクリスマスの魔法って感じでした。(今からもうワクワクが止まりません！)。2026年もどんなテーマで私たちを驚かせてくれるのか、今から期待が膨らみますね。

ハンジョンソンのフルーツヨーグルト餅、その魅力と楽しみ方

さて、話を「ハンジョンソン」のフルーツヨーグルト餅に戻しますね。昨年食べたメロンとマンゴーは、果実が本当に甘くてジューシーで、一口食べると口いっぱいに果汁が弾けました。ヨーグルトとの相性も抜群で、まさに「果物とヨーグルトの夢の出会い！」って感じでしたよ！🍓🍈🥭 ただ、イチゴは時期が少しずれていたのか、他の二つに比べて甘さが控えめだったのがちょっと残念でしたね。

このフルーツヨーグルト餅の最大のポイントは、なんといっても「鮮度」！購入後30分以内に食べるのがベストって言われるくらい、生果物を使ってるから日持ちしないんです。昨年、翌日に食べようとしたら、果汁が出てきちゃってお餅がべたついて、食感が台無しになっちゃいました…。だからこそ、購入したらすぐに、フレッシュなうちに味わうことを強くおすすめします!! 🚨

「ハンジョンソン」のフルーツヨーグルト餅は、単なるスイーツじゃなくて、ソウルのトレンドを肌で感じられる特別な体験なんです。2026年も、きっと新しいフレーバーや限定品が登場するはず！「ザ・ヒュンダイ・ソウル」を訪れるなら、ぜひ「ハンジョンソン」に立ち寄って、この絶品スイーツを味わってみてくださいね！🙌
