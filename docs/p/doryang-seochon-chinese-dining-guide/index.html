2026年も行列必至！西村「度量（ドリャン）」の絶品東坡肉とシェフの温かいおもてなし

人気番組『白と黒の泥棒たち』で話題のイム・テフンシェフの店「度量」。2026年5月の訪問に向けて、伝説の東坡肉や海老の茄子包み揚げの魅力を、過去の感動体験をベースにプレビューガイドとしてお届けします。

最近、ソウルの西村（ソチョン）界隈で一番熱い場所といえば、間違いなくここ「度量（ドリャン）」ですよね。

西村の街に佇む風格漂うエントランス

景福宮（キョンボックン）駅の近くを歩いていると、ひときわ存在感を放つ建物が見えてきます。

入り口で護衛のように立っている兵馬俑の像は、2026年の今も変わらず訪れる人を圧倒しています（これ、初見だと結構びっくりしますよねww）。黄金に輝く看板を目印に階段を上がれば、そこには本格的な中華の世界が広がっています。

箸で解ける柔らかさ！伝説の東坡肉

ここに来る人のほとんどが、この一皿を目当てにしていると言っても過言ではありません。

厚切りの豚バラ肉が、深いコクのあるソースを纏って登場するこの東坡肉は、まさに至高の一品です。お箸で簡単に切れるほどの柔らかさで、青菜と一緒に頬張ると脂の甘みが口いっぱいに広がります（笑） 2026年シーズンも、この完璧な火入れは健在ですね。

そして、東坡肉と並んで絶対に外せないのが、この「魚香茄子（ユーシャンチェズ）」です。サクサクの茄子の中に、ぷりっぷりの海老のすり身がこれでもかってくらい詰まっていて、ピリ辛のソースとの相性が抜群。これがまた、お酒にもよく合うんですよ 🍤

モダンで高級感あふれる店内の雰囲気

店内は洗練されたモダンな空間で、大切な日の食事にもぴったりの雰囲気です。

壁に飾られた黄金のロゴが、高級感をさらに引き立てています。非常に人気のお店ですが、イム・テフンシェフが各テーブルを回りながら料理の説明をしてくれるなど、温かいホスピタリティが感じられるのも「度量」が愛される理由（これが意外と嬉しいポイントなんです）。

テーブルに置かれた青白陶器の茶器も、食事の合間のひとときを優雅に演出してくれます。温かいお茶で口の中をリセットしながら、次の一皿を待つ時間はまさに贅沢そのもの。こうした細部へのこだわりが、体験全体の質をグッと引き上げてくれます。

締めの一杯に最適な絶品チャンポン

メインを堪能した後の締めには、具だくさんのチャンポンがおすすめです。

ムール貝やイカなどの海鮮が溢れんばかりに盛られたこのチャンポンは、見た目以上にスッキリとした辛さが特徴です。濃厚な料理の後にいただくこのスープは、まさに完璧なフィナーレ 🍜 リーズナブルなのにこのクオリティは、正直驚きしかありません。

茄子の揚げ物を半分に割ってみると、中から熱々の海老が顔を出します。この断面の美しさからも、シェフの丁寧な仕事ぶりが伝わってきますよね。一口食べるごとに幸せを感じられる、そんな繊細な味わいが2026年も多くの食通を虜にしています。

最後の一切れまで美しく盛り付けられた料理たちは、まさに芸術のようです。西村の歴史ある街並みを散策した後に、ここで究極の中華を楽しむ休日は、自分への最高のご褒美になるはず。整理券の確保は大変ですが、その苦労を遥かに上回る感動がここにはあります。
