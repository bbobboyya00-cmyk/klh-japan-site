35年の伝統が語る、ソウル東部・長安洞の隠れた名店

夜の帳が下りる頃、看板に灯がともると同時に店前には活気が溢れ出します。この「いかにも地元の名店」という佇まいを見ただけで、今夜のディナーが成功することを確信できるはずです。

ここの主役は何と言っても、希少部位のガブリサル（背肉）を使ったポッサム。口に運んだ瞬間に脂の甘みがじゅわっと広がり、気づけば溶けてなくなっているような感覚は、まさに未体験の衝撃です（笑）。

テーブルを埋め尽くすパンチャン（おかず）の数々も、一つひとつが主役級のクオリティ。特に自家製のムマルレンイ（切り干し大根の和え物）のポリポリとした食感は、肉の柔らかさをこれ以上ないほど引き立ててくれます 🍖

店内は常に地元の人々の熱気で満ちていて、これぞ韓国のローカル食堂というバイブスが最高です。観光客向けの店では味わえない、リアルなソウルの日常に溶け込むような体験がここにはあります（笑）。

採算度外視の無料サービスと、通が愛する屋外の特等席

驚くべきは、ポッサムを頼むと本格的なカルグクスかマッククスが「無料」で付いてくるという太っ腹すぎるシステム。サービスだからと侮るなかれ、野菜たっぷりのマッククスは専門店顔負けのボリュームと中毒性のある辛さが自慢です。

気候が良い時期なら、迷わず店先の赤いプラスチックテーブルを確保してください。ソウルの夜風を感じながら流し込むマッコリとポッサムの組み合わせは、もはや説明不要の至福のひとときです 🥢

ランチタイムなら13,000ウォン程度でこの豪華なセットを楽しめるというから、コスパの概念が崩壊しそうです（笑）。近隣の会社員たちが毎日通い詰めるのも、この価格とクオリティを見れば納得しかありません。

赤いオーニングが目印の店構えは、初めて訪れる人でもすぐに見つけられるはず。チャンハンピョン(長漢坪)駅から徒歩圏内なので、ソウル中心部から少し足を伸ばす価値は十分にあります。

余裕があれば、海鮮の出汁がしっかり効いた鍋も一緒にオーダーするのが通の楽しみ方。熱々のスープがお肉の旨味をさらに増幅させ、最後の一口まで飽きることなく完食させてくれます ✨

職人の技が光る、艶やかな肉の芸術

注文を受けてから丁寧に切り分けられるお肉からは、食欲をダイレクトに刺激する湯気が立ち上ります。木製の器に無造作ながらも美しく盛られた姿は、35年続く職人の矜持を感じさせます。

この艶、写真越しでも伝わるでしょうか。箸で持ち上げるだけで崩れそうなほど繊細で、噛むほどに溢れ出す肉汁は、まさに「ポッサムの終着駅」と呼ぶにふさわしい完成度です。

最後はシャキシャキの野菜がたっぷりのビビンククスで、口の中をさっぱりとリセット。この完璧なコース仕立ての食体験を終えた時、あなたはきっと「次は誰を連れてこようか」と考えているはずです。
